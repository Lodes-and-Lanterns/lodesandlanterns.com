# Lodes & Lanterns

Homepage for [lodesandlanterns.com](https://lodesandlanterns.com).

## Local Development

Serve the site locally with Deno:

```bash
deno run --allow-net --allow-read jsr:@std/http/file-server
```

Then open http://localhost:8000 (or whatever port is output if 8000 is taken).

## Deployment

This site is deployed via GitHub Pages via pushes to the `main` branch.
